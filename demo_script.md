# Demo script — LexEvidence Disclosure Agent

Target length: 3 to 5 minutes.

## 0:00 to 0:30 — Hook

Show a fictional AI-agent message that lacks AI disclosure and operator identity.

Message: organizations are deploying AI agents faster than they are building transparency, approval and audit controls.

## 0:30 to 1:00 — Input

Open the Streamlit app. Select `example_critical_risk.json` from the sidebar.

Explain: all data is synthetic. No real client files, no real emails, no confidential information.

## 1:00 to 2:00 — Audit

Show the compliance score, overall risk and 10-principle table.

Point to failed principles: missing AI disclosure, missing human approval, weak audit trail and reserved activity risk.

## 2:00 to 3:00 — Risk matrix

Open the risk matrix tab. Show message-level risk and reasons.

Explain: the tool is an auditor, not an autonomous decision-maker.

## 3:00 to 4:00 — Report and policy output

Show the recommended disclosure note, internal policy snippet and Markdown report download.

## 4:00 to 4:30 — Responsible AI close

State: the MVP runs locally, uses deterministic scoring, uses synthetic data only, and does not provide legal advice.

## Optional 10 to 15 seconds

Show VS Code and GitHub workflow. Mention GitHub Copilot assistance only if it was actually used during implementation or test creation.

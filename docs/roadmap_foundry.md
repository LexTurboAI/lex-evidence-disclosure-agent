# Microsoft Foundry roadmap

This repository currently runs as a local-first Creative Apps MVP.

## Current version

- Python + Streamlit
- Local parser
- Deterministic 10-rule audit engine
- Local Markdown report generation
- Synthetic JSON logs
- No Azure runtime dependency
- No Microsoft Foundry runtime dependency

## Planned future upgrade

A future version may add Microsoft Foundry / Foundry IQ as an optional reasoning layer:

1. Use Foundry as a second-opinion reasoning module.
2. Ground explanations in the Lex Turbo Agent Disclosure Standard and selected regulatory materials.
3. Compare deterministic rule output with model-assisted analysis.
4. Preserve local deterministic scoring as the primary auditable baseline.

## Explicit disclosure

Microsoft Foundry / Foundry IQ is planned, not used in this version.

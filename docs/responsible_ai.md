# Responsible AI

LexEvidence Disclosure Agent is designed as a bounded audit demonstration.

## Principles

1. **Synthetic data only** — the repository includes only fictional JSON logs.
2. **No legal advice** — the output is an audit-style demonstration, not a legal opinion.
3. **Human-in-the-loop** — recommendations require review by a qualified human before implementation.
4. **No autonomous actions** — the tool does not send emails, make decisions or modify external systems.
5. **Transparent scoring** — each principle returns a score, evidence and justification.
6. **Bounded scope** — the MVP checks 10 disclosure and auditability principles only.
7. **Privacy by design** — the default version runs locally and does not require cloud processing.

## Limitations

The tool cannot determine real legal compliance by itself. It helps identify missing disclosures, weak approval controls and poor audit trails in synthetic AI-agent logs.

# Privacy

This MVP is designed for local-first processing.

## Data policy

- Use synthetic logs only.
- Do not upload real client data, legal case materials, private emails, medical data or confidential information.
- The default Streamlit app processes the selected JSON file locally.
- The repository contains no API keys, secrets or external service credentials.

## Confidentiality warning

This project is a hackathon prototype. It is not intended for processing privileged, sensitive or production data. Any real-world deployment would require a separate security, privacy and legal review.
